Get the manual on paper

The full SWI-Prolog manual is also available as a print-on-demand book. This comprehensive manual gives an overview of the system and how to set-up programming projects with it, it describes the built-in predicates, module system, coroutining, threads, the Constraint Handling Rules library, the foreign language interface and how to generate runtime applications.

Clicking the cover page at the left brings you to google books, which shows the places where you can order the book.
